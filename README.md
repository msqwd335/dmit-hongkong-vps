## 三条线路分别是什么逻辑

### HKG.Pro：Premium 系列，CN2 GIA 精品线路

这是 DMIT 香港的旗舰线路。电信双向 CN2 GIA，联通走 AS9929 精品路由，移动 CMI 直连。三网回程都走高端线路，对国内延迟优化最彻底。

用下来最直观的感受是：香港到国内主要城市，正常情况延迟在 30-50ms 以内。这个数字在同类机房里属于顶格水平。

带宽是什么情况？常规套餐标注 1Gbps，但更准确的说法是这个带宽是共享的，实际跑出来的速度取决于负载时段。特供款（MongKok / VICTORIA）带宽分别是 150Mbps 和 500Mbps，相对稳定一些。

代价是贵。常规套餐最低 $39.9/月，流量只给 500GB。建站流量大的场景确实容易不够用。

IP 被墙有免费换的机制——每 15 天可以换一次，超出的话 $5/次。

### HKG.EB：Eyeball 系列，CMI 优化线路

移动双向 CMI 直连，联通和电信回程直连，去程绕日本 NTT。

说实话这条线路的定位有点尴尬，比 T1 贵，比 Pro 便宜，但线路优先级和 Pro 差一级。适合什么人？主要用移动网络的用户，或者预算有限但又不想完全放弃国内优化的场景。

带宽给得很大方。TINYv2 就是 1Gbps，往上走的套餐直接给 2G 甚至 4Gbps。流量也比 Pro 多很多，4 核 4G 的机器给 8TB/月，这个规格在香港优化线路里不算常见。

### HKG.T1：Tier 1 系列，国际线路 + 真正的大带宽

这条线路面向国际业务。接入了 RETN，香港到欧洲的延迟非常短；对大陆方向没有特别优化，电信联通路由表现一般。

但它是三条线路里带宽上限和流量最夸张的一条。GIANT 套餐 8 核 16G 配 128TB 月流量，$199.9/月。WEE 入门款年付只要 $36.9，1T 流量，用来跑一些纯国际业务非常够用。

还有一点值得提：T1 系列流量超出不停机，降速继续用。WEE 套餐限速到 50Mbps，不会突然断线。

## 到底哪条线路适合你

三条线路不是越贵越好，关键在用途。

**场景一：用户主要在中国大陆**

优先考虑 Premium。电信从北京、上海过去延迟普遍在 40ms 以内，高峰期跑 150Mbps 没什么问题。建站、游戏加速、做落地机都可以。流量不够用可以直接升套餐，或者从 VICTORIA 这个特供款入手，年付 $298.88，配置比常规 TINY 厚实。

问题是流量真的不多，500GB 一个月对高流量业务是个限制。

**场景二：预算有限，但还是想要一定的国内速度**

Eyeball 是这里的答案。移动走 CMI 直连，效果比较稳；电信联通高峰期表现不如 Pro，但比纯国际线路好。TINYv2 才 $17.9/月，给了 1Gbps 带宽和 1TB 流量，性价比在这个价位段算高的。

用过的人对这条线路的移动用户体验评价通常不错，电信联通的话则看峰值时段。

**场景三：做国际业务，或者主要流量来自海外**

T1 直接选。$6.9/月起，流量给到 2TB，WEE 年付款更是 $36.9 就能拿到 1T 流量。香港到欧洲通过 RETN 延迟很低，做跨境应用落地非常够用。对大陆延迟不在乎就别为 CN2 GIA 多花三四倍钱。

## 几个用起来容易踩的坑

买之前有几件事值得先想清楚：

1. **流量计算方式**：T1 系列是单向计费（只算出流量），Pro 和 EB 系列是双向计算。同样标注 1TB，实际可用量差一倍——这一点官网写了但很多文章没提。
2. **带宽共享**：所有套餐的带宽都是共享的，不是独享。1Gbps 是上限，实际跑多少取决于同节点负载。EB 系列给的带宽上限高，峰值更容易跑满。
3. **IP 被墙**：Premium 系列提供每 15 天免费换 IP 一次的政策，T1 和 EB 系列需要另外收费。如果担心 IP 稳定性，Pro 系列这个政策算是额外的保障。
4. **退款**：DMIT 有 3 天内退款政策，但条件比较严格，要求未违反使用条款。想试水的话建议先从入门款开始，别一上来就买高配。

付款支持支付宝和微信，国内用户操作没障碍。

## 常见问题

**Q：DMIT 香港大带宽 VPS，哪个套餐带宽最大？**

Eyeball 系列给的带宽上限最高，MICROv2 及以上标注 4Gbps 共享带宽；T1 系列标注 1Gbps 并发但流量天花板大得多（最高 128TB/月）。如果追求带宽峰值能跑高，EB 系列更合适；如果追求月流量大，T1 更合算。

**Q：香港 VPS 大带宽能不能用于直播推流？**

可以。Premium 和 Eyeball 系列的香港节点延迟低、带宽稳定，用作推流落地机没什么问题。建议选 Premium 保证国内回程质量，VICTORIA 特供款（500Mbps/800GB）性价比相对合理。

**Q：DMIT 香港 T1 和 Eyeball 有什么本质区别？**

线路完全不同。T1 是国际线路，对中国大陆方向没有优化，适合海外访问；Eyeball 走移动 CMI 直连，对大陆有一定优化，适合国内用户访问。价格上 T1 便宜很多，但如果用户在国内，T1 体验会差一截。

**Q：流量用完会直接停机吗？**

T1 系列不会，超出后限速继续用（WEE 款限速到 50Mbps）。Premium 和 Eyeball 系列超出后的处理方式建议购买前通过官方客服确认最新政策，以官网说明为准。

**Q：DMIT 香港 VPS 支持解锁流媒体吗？**

全系标配原生 IP，实测可以解锁香港区 Netflix、YouTube Premium、ChatGPT 等服务。IP 解锁情况是动态变化的，DMIT 不以此为购买承诺，建议以实测结果为准。
